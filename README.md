## 🚀 快速开始

### 在线使用

#### 方式一：开发模式（推荐）

```bash
npm install
npm run dev
# 访问 http://localhost:5173
```

#### 方式二：生产构建

```bash
npm install
npm run build
# 使用任意 HTTP 服务器打开 dist/ 目录
```

### 命令行使用

```bash
npm install
node cli.js <markdown-file> [options]
```
